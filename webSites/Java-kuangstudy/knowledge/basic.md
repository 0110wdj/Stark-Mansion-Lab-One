## 01 注释

```Java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

```Java
public class Main {
    public static void main(String[] args) {
        // 单行注释
        System.out.println("Hello world!");
        /*
        多行注释
         */

        // JavaDoc:文档注释 /** */
        /**
         * @Description Hello word
         * @Author Snofly
         */

    }
}
```
